Build a complete full-stack web application called "ChronoMind: The Neuromorphic Future Simulation Engine".

PROJECT VISION

ChronoMind is inspired by the concept of Dr. Strange using the Time Stone to explore millions of possible futures before making a decision.

The application MUST NOT claim to actually predict the future. Instead, it should simulate possible future outcomes using a brain-inspired neuromorphic architecture based on memory networks, associative reasoning, probabilistic branching, and adaptive learning.

The experience should make users feel like they are exploring alternate realities through a mystical neural interface.

The project must be hackathon-ready, visually impressive, and fully functional.

========================================================
CORE USER FLOW
========================================================

1. User enters:
   - Current Situation
   - Possible Decision(s)
   - Optional Context

Example:

Current Situation:
"I am a first-year computer science student."

Decision Options:
- Learn AI
- Learn Cybersecurity
- Learn Web Development

2. System activates relevant memories and concepts.

3. System generates hundreds of possible future branches.

4. System evaluates probabilities.

5. System visualizes branching timelines.

6. System presents:
   - Most Likely Future
   - Top 5 Futures
   - Rare Futures
   - Probability Scores
   - Explanations

7. User can compare alternate futures.
